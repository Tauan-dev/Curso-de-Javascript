Metodos de String

  • trim
    • remove tudo que não é string
  • padstart
    • insere valores no incio de uma string passados por parametros, onde variavel.padStart(tamanhoStringFinal,valor)
  • split
    • divide uma string por um separador, e retorna um array
  • join
    • faz o processo inverso ao split, unindo o array por um separador de dados qualquer
  • repeat
    • repete uma string de acordo com valor de um parametro

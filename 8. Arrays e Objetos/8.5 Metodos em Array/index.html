Metodos em array

Metodos de Arrays

  • push and pop
    • O metodo push adiciona elementos no fim do array, e o pop remove
  • shift e unshift
    • funciona da mesma forma que o push and pop, porém adiciona e remova os primeiros elementos do array
  • indexOf e lastIndexOf
    • puxa no array a primeira posição do parametro e a ultima
  • slice
    • retorna um array a partir de outro, que é determinado por parametros, que seriam o inicio e o final do array
  • forEach
    • funciona como um loop (for ou while), iterando cada elemento do array
  • includes
    • verifica se o array tem um determinado elemento
  • reverse
    • inverte um array, não importando o tipo de dado
